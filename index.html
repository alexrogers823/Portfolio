About Me

Musician turned coder. I was born and raised in Minneapolis, MN, and graduated from Emory University in 2016 with a double major in Finance/Music. I then worked as a Financial Analyst for two years before wanting to dive into code.

My coding journey started as me learning Python before graduating, which spiraled into spending several hours at Starbucks doing code, and ultimately going to DigitalCrafts to expand my knowledge. Coding is life. I can't go a day without it.

Coding is Music

Programming is both an art and a science, with many creative elements. Constructing code is similar to analyzing and composing music. It uses the same part of the brain. Therefore, learning to code is - in essence - learning a new instrument. This is why several musicians can relate to software development
